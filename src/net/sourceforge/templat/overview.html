TemplAT is a text-file template engine. This documentation is intended for developers interested in the internal structure of the TemplAT library. Normal users of TemplAT will only need to use the {@link net.sourceforge.templat.Templat net.sourceforge.templat.Templat} class.

The library consists of a lexer ({@link net.sourceforge.templat.lexer} package) and a parser ({@link net.sourceforge.templat.parser} package), as well as a special lexer and parser just for expressions ({@link net.sourceforge.templat.expr} package), and a parsing context handler ({@link net.sourceforge.templat.parser.context} package).
